# Twemoji CSS Lib
Twemoji CSS Library for Static Sites which is used by Cheeter Technologies and many personal Projects

###### This project isnt entirely made by me, most of it is made by the creator(s) or a/some user(s) of maxcdn.com, just thought to share this here as it was very hard for me to find a working Twemoji Library for Static and non Static Sites.

###### Check if maxcdn.com is online before using this Project, if its offline then this Project will break too.
